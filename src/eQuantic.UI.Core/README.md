# eQuantic.UI.Core

Core abstractions and interfaces for the **eQuantic.UI** framework.

## Overview

This package provides the fundamental building blocks for eQuantic.UI:

- `IComponent`: The base interface for all components.
- `StatefulComponent`: Base class for components with state.
- `StatelessComponent`: Base class for pure rendering components.
- `HtmlElement`: Abstraction for native HTML elements.
- `RenderContext`: Context provided during the build process.

## Installation

This package is usually installed automatically via the `eQuantic.UI.Sdk`.

```bash
dotnet add package eQuantic.UI.Core
```

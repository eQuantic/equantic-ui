# eQuantic.UI.CLI

Command-line interface tools for **eQuantic.UI** developers.

## Overview

The CLI provides helpful commands for developing with eQuantic.UI:

- **Scaffolding**: Create new projects or components (`equi new`).
- **Dev Server**: Run a development server with enhanced logging.
- **Diagnostics**: Analyze project structure and configuration.

## Installation

```bash
dotnet tool install -g eQuantic.UI.CLI
```
